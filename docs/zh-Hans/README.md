如何优雅的阅读本文档？

只需要运行sheel脚本，然后打开浏览器访问 `http://localhost:3000` 即可

- Linux  
`bash docs/run.sh`

- Winodws  
    - python 3  
        `cd docs && python -m http.server 3000`
    - python 2  
        `cd docs && python -m SimpleHTTPServer 3000`
    - docsify  
        `docsify serve docs`


由 [docsify](https://github.com/docsifyjs/docsify) 提供文档支持
